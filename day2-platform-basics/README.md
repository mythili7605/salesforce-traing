# Day 2 - Platform Basics

## 1. What is Salesforce Platform?

Salesforce Platform is a cloud-based platform used to build applications, manage customer data, automate business processes, and improve customer relationships. It provides tools for customization, automation, reporting, and app development without needing much hardware setup.

---

## 2. Explain

### App
An app in Salesforce is a collection of tabs, objects, and features grouped together for specific business purposes.

### Object
An object is a database table used to store data in Salesforce.
Examples:
- Standard Object: Account, Contact
- Custom Object: Student, Employee

### Tab
A tab is used to access objects, dashboards, reports, and other features in Salesforce.

---

## 3. Difference: Configuration vs Coding

| Configuration | Coding |
|---|---|
| Uses click-based tools | Uses programming languages |
| Easy to implement | Requires coding knowledge |
| Faster development | More flexible customization |
| Example: Flow, Validation Rule | Example: Apex, Lightning Components |
